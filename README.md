# Express-Note-Taker

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Description:

A digital note pad and/or task creater that allows you to create/write notes with a title and text, read notes that have already been created and delete notes you no longer want and/or need. 

## Usage:
Enter note title and text. You can then click the save note button to save it to list or clear form to start over. To delete a saved note simply click the trash can icon next to that note.
